Object Detection on NXP 8MMNavQ Kit Starter Kit

The Google Coral Camera attached to the NavQ board is able to detect objects. By using SSD, we only need to take one single shot to detect multiple objects within the image, while regional proposal network (RPN) based approaches such as R-CNN series that need two shots, one for generating region proposals, one for detecting the object of each proposal. Thus, SSD is much faster compared with two-shot RPN-based approaches.
